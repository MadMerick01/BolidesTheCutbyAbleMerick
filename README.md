# Bolides: The Cut

## BeamNG API verification
All BeamNG APIs used in this repo **must** be validated against the API dump in `docs/beamng-api/raw/`.
See `docs/CODEX_RULES.md` and `docs/beamng-api/README.md` for the required workflow and lookup tooling.

## Repository layout (high level)
- `docs/beamng-api/`: BeamNG API dump, indexes, and topic notes
- `lua/`: gameplay and system logic
- `ui/`: UI assets
